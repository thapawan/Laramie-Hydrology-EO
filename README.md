### Laramie Water Cycle – Earth Observation Study
This repository contains the data and scripts for the Laramie Hydrologic Story Map. This project demonstrates the integration of in-situ field observations (360° Virtual Field Trips) with NASA Earth Observation (EO) datasets.

🛠 Tools Used
Google Earth Engine (GEE): Time-series analysis of Landsat & Sentinel-2.

NASA VFT (ASU): Interactive 360° environment for spatial context.

Medial Axis Transform (MAT): Used for extracting river centerlines in the Laramie Basin.

📂 Repository Contents
/notebooks: Python scripts for calculating NDVI and NDWI.

/assets: Field photographs and site metadata.

/docs: Project abstract and the 'HydroMorphNet' framework overview.
